# Cozy Girls Café Website

## Project Description

A girls-only coffee shop website designed to provide a cozy,
creative, and relaxing atmosphere.

The platform allows users to:
- explore activities
- reserve workshops
- browse gallery photos
- order food and drinks
- add items to cart
- login and logout

---

# Technologies Used

- React
- TypeScript
- Vite
- Tailwind CSS
- React Router DOM
- Context API

---

# Main Features

## Authentication
- Login page
- Sign Up page
- Logout button
- Loading spinner

## Activities
- Painting
- Pottery
- Candle making
- Journaling
- DIY crafts

## Menu
- Coffee
- Shakes
- Croissants
- Desserts

## Cart System
Users can:
- add items to cart
- remove items
- view total items

## UI/UX
- Cozy café design

- Animations
